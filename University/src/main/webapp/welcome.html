The University database consists of various tables for storing and performing operations on student and faculty.The above menu bar shows four tabs where user can navigate from one page to another based on the user requirement.

  1. Home
  2. Display
  3. Insert
  4. Modify
  5. Delete

Home: This is the present page used for welcoming the university database and there he can perform various operations.

Display: This display the content of Instructor, department, and courses tables.

  • Program should ask the table name to display when selecting the display data option from the main menu. Then display the content of the selected table.

Insert: This is used add the functionality to insert record to the department table.

  • User should be able to select the option to insert data from the main menu
  • User should be asked to enter each column data one after the other and once the final column data is inserted, it should insert the data to the department table.

Modify: This is used to add a main menu item to modify the budget of a given record of the department table

  • User should be able to select the modify budget menu option from the main menu.
  • The user is presented to enter the department name and the program displays the content of that tuple and asks from the user the new value for the department budget. The program then updates the budget of that department

Delete: This is used to add a main menu item to delete a department from the department table

  • User should be able to select the delete department menu option from the main menu.
  • The user is presented to enter the department name and the program displays the information of that department. Then the program asks the user “Are you sure?”
1. If the user selects yes-> delete that department from the department table 2. If the user selects no -> return to main menu
